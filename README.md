# iComrate

A web application to share and book avaiable resources 

Technologies

 * [ExpressJS](https://expressjs.com/): Node.js web framework


### Usage

Install library:

```javascript
yarn
```
Then run for windows
```javascript
copy .env.example .env
```
Or for Linux/Mac
```javascript
cp .env.example .env
```

Run Express server on port 8000:

```javascript
yarn start
```
Run test:

```javascript
yarn test
```
